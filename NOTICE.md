## Attribution

The original emulator implementation (RISC-V RV32IMA) was created by Charles Lohr.
This project includes modified, refactored, and reorganized versions of that code.

Key changes made in this project include:

* Refactoring from multi-file / macro-heavy structure into a simplified layout
* Consolidation into a minimal runnable structure (single command execution)
* Cleanup and restructuring for readability and portability
* Preparation for cross-language porting (e.g. Rust)
